Farben

Farben können in CSS auf verschiedene Arten angegeben werden:

  • Farbnamen: color: blue;
  • Rot/Grün/Blau: color: rgb(0, 0, 255);
  • Hex-RGB: color: #0000ff;

Bei RGB kann man die Werte entweder in dezimaler (von 0 bis 255) oder in hexadezimaler (von 00 bis ff) Schreibweise angeben.
Zur Veranschaulichung hier ein paar verschiedene Zahlensysteme (dezimal, hexadezimal, binär):

{% for d in dez %} {% endfor %} {% for h in hex %} {% endfor %} {% for b in bin %} {% endfor %}
dez{{ d }}
hex{{ h }}
bin{{ b }}

Farben mischen

Man unterscheidet zwischen additiver Farbmischung (Licht) und subtraktiver Farbmischung (z.B. Aquarellfarben).
Bei der Mischung mittels RGB handelt es sich um additive Farbmischung,
entsprechend ergibt rgb(255, 255, 255) bzw. #ffffff die Farbe weiß.

Wer sich für Farbtheorie interessiert, kann auf Wikipedia: Farbmischung eine gute Übersicht erhalten
oder mit dem englischen (aber leicht zugänglichen) Guide des NASA Projekts DEW tiefer einsteigen.
